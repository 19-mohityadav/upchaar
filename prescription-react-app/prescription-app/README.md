# Dr. Debasish Sinha — Prescription Template (React)

A pixel-faithful React recreation of the Upchar Health prescription letterhead.

## Tech Stack
- React 18
- Vite
- CSS3 (Google Fonts: Playfair Display + Source Sans 3)

## Getting Started

```bash
npm install
npm run dev
```

Then open http://localhost:5173

## Build for Production
```bash
npm run build
```

## Design Notes
- Colors: Red (#E63946), Teal (#1A9E9E), Dark (#1e1e1e)
- Fonts: Playfair Display (headings/logo), Source Sans 3 (body)
- Layout: A4 portrait (794×1123px)
- All SVG icons are hand-coded (no external icon libraries)
- Upchar Health logo recreated as inline SVG
- Watermark rendered as low-opacity background element
- Corner accent stripes match original diagonal design
